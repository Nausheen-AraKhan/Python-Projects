# Caesar Cipher 🔐

A simple Python implementation of the Caesar Cipher encryption and decryption technique.

## 📖 About the Project

The Caesar Cipher is one of the oldest and simplest encryption methods. It works by shifting each letter in a message by a fixed number of positions in the alphabet.

For example, with a shift of **3**:

* A → D
* B → E
* C → F

This project allows users to:

* Encrypt messages
* Decrypt messages
* Choose their own shift value
* Work with uppercase and lowercase letters

## 🚀 Features

* Encrypt text using Caesar Cipher
* Decrypt encrypted text
* User-defined shift value
* Easy-to-understand Python code
* Beginner-friendly project
