# 🏓 Pong Game

A classic **Pong Game** built using **Python** and the **Turtle** graphics library. This project recreates the iconic arcade game where two players control paddles to bounce a ball back and forth. The first player to miss the ball gives a point to the opponent.

## 🎮 Features

- Two-player gameplay
- Smooth paddle movement
- Ball collision with paddles and walls
- Automatic ball speed increase after each successful hit
- Real-time score tracking
- Simple and clean user interface

## 🛠️ Built With

- Python 3
- Turtle Graphics

## 📂 Project Structure

```
Pong-Game/
│── main.py          # Main game file
│── paddle.py        # Paddle class
│── ball.py          # Ball class
│── scoreboard.py    # Scoreboard class
└── README.md
```

## 🎯 Controls

| Player | Up | Down |
|--------|----|------|
| Left Paddle | A | Z |
| Right Paddle | ↑ | ↓ |
