# 🐢 Turtle Crossing Game

A fun arcade-style game built using **Python** and the **Turtle graphics** module as part of **Dr. Angela Yu's 100 Days of Code: The Complete Python Pro Bootcamp**.

The objective is simple: **Help the turtle cross the road while avoiding the moving cars!** Every successful crossing increases the game's difficulty by making the cars move faster.

---

## 🎮 Gameplay

- Control the turtle using the **Up Arrow (↑)** key.
- Avoid colliding with the moving cars.
- Reach the top of the screen to level up.
- Each new level increases the speed of the cars.
- The game ends when the turtle collides with a car.

---

## ✨ Features

- 🐢 Player-controlled turtle
- 🚗 Randomly generated moving cars
- 📈 Increasing difficulty with each level
- 🏆 Level tracker
- 💥 Collision detection
- 🎯 Game Over screen

## 🛠️ Built With

- Python 3
- Turtle Graphics
- Object-Oriented Programming (OOP)
