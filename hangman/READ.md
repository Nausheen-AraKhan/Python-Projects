# Hangman Game 🎮

A classic Hangman game built using Python, where players try to guess a hidden word one letter at a time before running out of lives.

## 📖 About the Project

Hangman is a popular word-guessing game. The player must guess the letters of a hidden word. For every incorrect guess, a part of the hangman is drawn. The game ends when the player either guesses the word correctly or runs out of lives.

This project was built to practice Python fundamentals such as loops, conditionals, lists, functions, and working with multiple files.

## 🚀 Features

* Random word selection
* Letter-by-letter guessing
* Lives system
* Visual hangman stages
* Win and lose conditions
* Beginner-friendly code structure

## 🎯 Game Rules

1. The computer selects a random word.
2. Guess one letter at a time.
3. Correct guesses reveal the letter's position(s).
4. Incorrect guesses reduce your remaining lives.
5. Guess the entire word before your lives reach zero to win.
